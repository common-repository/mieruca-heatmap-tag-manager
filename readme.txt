This is the plugin to introduce the tag of the free heatmap service "Mieruca Heatmap" easy.

== Description ==

無料で使えるヒートマップツール、『ミエルカヒートマップ』のタグ設置が簡単にできるプラグインです。

= ミエルカヒートマップとは？ =

SEO対策サービス『<a href="https://mieru-ca.com/" target="_blank">ミエルカ</a>』の姉妹プロダクト。
以下の3種類のヒートマップから、売上、SEO流入、CVR改善を見える化し、UI/UX改善へのヒントをご提供します。
    - スクロールヒートマップ: Googleアナリティクスではわからない「ページ内での離脱箇所」を可視化
    - クリックヒートマップ: ページ内でよくクリックされている場所を可視化、コンバージョンを妨げているクリック場所がないか確認可能
    - アテンションヒートマップ: 訪問者がどのコンテンツに注目しているかを可視化、CV改善へのヒントに活用可能
- 複数のアクセス端末 (PC 、スマートフォン、タブレット) 毎に解析可能、レスポンシブ時代の新たな分析にご活用ください。

より詳しい機能は<a href="https://mieru-ca.com/heatmap/" target="_blank">ミエルカヒートマップの公式サイト</a>でご確認ください。

== Installation ==

1. (すでにアカウントをお持ちの方はそのまま3.に進んでください) ミエルカヒートマップのWebサイト (https://mieru-ca.com/heatmap/) のお問い合わせフォームに必要事項を入力し、無料アカウントを発行してください。
2. ミエルカヒートマップ内にて初期設定を進め、ご登録いただいたサイト専用の解析用タグを発行してください。
3. プラグインの入力フォームに発行済みの解析用タグを貼り付け、保存ボタンをクリックしてください。
4. 設定が完了したタイミングから計測がスタートします。翌日以降、ミエルカヒートマップより計測データを閲覧ください。
